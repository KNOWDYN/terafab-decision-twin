# Browser-only GitHub Pages deployment

1. Open `https://github.com/KNOWDYN/terafab-decision-twin` in a browser.
2. Upload the contents of this package to the repository root, preserving the `docs/` folder.
3. Open repository **Settings**.
4. Open **Pages** in the sidebar.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select:

```text
Branch: main
Folder: /docs
```

7. Click **Save**.
8. After GitHub publishes the site, open:

```text
https://knowdyn.github.io/terafab-decision-twin/
```

9. Confirm that the GitHub icon opens the repository.
10. Confirm the PWA action reads `Keep this site`.
