Model architecture

Physics, finance, readiness, and public legitimacy share one calculation path.

The package validates evidence-coded scenario JSON, expands a time axis, computes equation modules, and returns scalar, vector, and matrix outputs for decision review.

Input

Evidence-coded scenarios

Material assumptions carry value, unit, status, source reference, confidence, and notes.

Kernel

Deterministic solver

The core engine computes coupled consequences across infrastructure, operations, economics, and policy.

Output

Gate matrix

Decision gates expose blockers instead of hiding risk inside a single aggregate score.

Core equations

The public site shows the logic without pretending to be the full source code.

The authoritative implementation remains in the repository; this page gives leadership the equations that matter first.

Thermal removal

Q_remove = P_IT + P_process + P_aux − W_useful − ΔU

Converts facility power into cooling burden.

Exergy destruction

X_destroyed = T0 × S_gen

Expresses irreversibility as lost work potential.

Water coupling

m_water = Q_remove / (c_p × ΔT × η_cooling)

Connects heat rejection to withdrawal and consumption stress.

Net value exposure

V_net = Y × V_gross − C_energy − C_water − C_delay − C_risk

Links yield and constraints to economic outcome.

Policy
