Getting started

Run the Terafab Decision Twin locally after uploading or cloning the repository.

The GitHub Pages site is the public front door. The repository is the runnable package.

CLI quickstart

python -m terafab_decision_twin.cli validate scenarios/baseline_2026.json
python -m terafab_decision_twin.cli run scenarios/terawatt_stress_2026.json
python -m terafab_decision_twin.cli report scenarios/multi_year_2026_2030.json

Browser-only deployment

  1. Upload the repository contents to GitHub.
  2. Upload this package so the repo contains /docs.
  3. Open Settings → Pages.
  4. Choose Deploy from a branch.
  5. Select main and /docs.
  6. Save and open the published URL.
