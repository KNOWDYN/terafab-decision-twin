Deploy from docs

A static GitHub Pages site with no backend and no build server requirement.

Publish from the repository settings by selecting Deploy from a branch, the main branch, and the /docs folder.

Upload or merge repository

Keep the `docs/` folder at the repository root.

Enable GitHub Pages

Repository Settings → Pages → Deploy from a branch → main → /docs → Save.

Verify public URL

Open `https://knowdyn.github.io/terafab-decision-twin/`, `sitemap.xml`, `llms.txt`, and `manifest.webmanifest`.

Install as PWA

Use the browser install prompt or Add to Home Screen on supported devices.
