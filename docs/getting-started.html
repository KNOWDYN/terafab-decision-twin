Deploy and use

Publish the site, install the package, and run scenarios.

This page is for maintainers publishing GitHub Pages and analysts using the Python package locally, in Jupyter, or in Colab.

Publish website

GitHub Pages from /docs.

Repository Settings → Pages → Deploy from a branch → main → /docs → Save.

https://knowdyn.github.io/terafab-decision-twin/https://knowdyn.github.io/terafab-decision-twin/sitemap.xmlhttps://knowdyn.github.io/terafab-decision-twin/llms.txt

Install package locally

Use editable install while developing scenarios.

python -m venv .venv. .venv/bin/activatepython -m pip install -e .terafab version

Run from CLI

Validate first, then simulate.

terafab scenario validate scenarios/baseline_2026.jsonterafab simulate scenarios/baseline_2026.json --output runs/baseline.json --report runs/baseline.mdterafab export scenarios/baseline_2026.json runs/baseline_bundle

Use in Jupyter

Minimal notebook cell.

Load a scenario, run the engine, inspect summary outputs and gates.

from pathlib import Path
from terafab_decision_twin import load_scenario, run_scenario

scenario = load_scenario(Path("scenarios/baseline_2026.json"))
result = run_scenario(scenario)
result["summary"], result["gates"]
