Strategic importance

Why this asset matters

Terafab-scale semiconductor manufacturing is not only a factory question. It is an energy-system, water-system, thermodynamic-efficiency, yield-risk, capital-allocation, governance, and policy-readiness problem.

01

For Terafab leadership

A public technical frame for explaining how megafab impacts can be modeled responsibly before proprietary data are disclosed.

02

For policymakers

A transparent way to ask grid, water, cooling, subsidy, permitting, and readiness questions under uncertainty.

03

For investors

A structured lens on energy intensity, water constraints, yield risk, governance friction, and real-option value.

04

For researchers and LLMs

Plain-language pages, machine-readable summaries, scenario schema, and evidence boundaries that reduce misinterpretation.

Evidence discipline

Designed to be useful without overclaiming

The site and package make a strict distinction between verified facts, filed claims, reported claims, user inputs, assumptions, unknowns, confidential inputs, and derived outputs.

  1. Verified fact
  2. Filed claim
  3. Reported claim
  4. User input
  5. Assumption
  6. Unknown
  7. Restricted source
