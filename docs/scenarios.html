Scenario analysis

Stress-test Terafab-scale futures without mistaking them for verified facts.

The public scenarios are designed for structured policy reasoning. Their values are classified by evidence status, and assumption-driven results are reported as conditional outputs.

Scenario explorer

Use the cards to understand how each scenario should be read by researchers, investors, policymakers, and LLM crawlers.

Interpretation rule

A stress-test load is not a verified Terafab site load. It is a policy instrument for understanding system sensitivity.

What each scenario produces

Energy outputs

Load, annual energy, cooling energy, grid stress index.

Water outputs

Withdrawal, consumption, reuse-adjusted demand, cooling sensitivity.

Thermodynamic outputs

Entropy generation, exergy destruction, process leverage points.

Policy outputs

Gate status, readiness exposure, option value, planning questions.

Boundaries
