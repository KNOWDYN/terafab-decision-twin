# Stakeholder Decision Surfaces

A stakeholder decision surface is a compact synthesis of uncertainty, strategic interaction, reduced-order dynamics, and validation-readiness checks.

The included helper `strategic_simulation.build_stakeholder_decision_surface()` produces four views:

```text
board_view
  gate-failure probability, dominant gate risks, recommended phase posture

investor_view
  cost distribution medians, strategic conflict index

policy_view
  water/wastewater/legitimacy risk indicators and range flags

research_view
  run counts, game profile count, ROM steps, validation level
```

The surface is a screening artifact. It is not investment advice, official project validation, or a permitting conclusion.
