# Restricted Sources Exclusion

This repository does not include restricted, confidential, proprietary, or non-redistributable source documents.

The public package may encode admitted facts, public model structure, equations, schemas, examples, and evidence gates. It must not embed restricted report text, proprietary operating data, confidential inputs, or any statement implying Terafab endorsement, authorization, adoption, acquisition interest, or employee participation.

Authorized reviewers with separate access to private materials should keep those materials outside the repository, for example in `local_sources/`, and should not commit them.
